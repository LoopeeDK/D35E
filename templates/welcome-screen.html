Welcome to 3.5E SRD implementation for Foundry VTT! It was created from love for one of the best versions of most popular Role Playing Game and aims to provide full 3.5E SRD implementation and system flexible enough for anything else.

Happy GMing,
Rughalt

News

Linked Items

Starting with version 0.94.2, items can now finally link to other items - they will be added and removed along the source item! This allows to create a lot of different and complex effects!

Auras

Version 0.95 introduces Auras! Auras allow characters to affect nearby tokens - pass changes and combat modifiers! Now Paladin can really create an Aura of Courage!

Auras are a work in progress! They may break and contain bugs!
